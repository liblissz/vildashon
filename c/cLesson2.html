Various Parts

  • #include
    •  this is a preprocessor directive that is used to include external files (header files) into your source code.For example stdio.h,stdlib.h
  • main()
    •  This is a funtion that serves as an entry point to every C program.
  • Comments
    •  Comments are of two types, sinle-line(/*every line*/ ) and multi - line//BEGINS THIS WAY comments.They help the programmer make notes aabout a funtion or some blog of code this enhances Understanding of the code. They are often ignored by the compiler
  • Printf()
    •  this funtion is used when ever a c programmer wants to display some output unto the console. In this case we are printing the text "Hello world !"
    Note :In C text must alwys be printed indouble codes
  • return 0 ;
    •  This statement is used to indicate succeful exxecution of the program. If a C program returns any other value apart form "0", it indicates error in the code at one point
  • Statements
    •  Statements are a fundamental units of execution .theyy instruct the computer to perform specific actions. most staments in c end with a semi colon(;) an examples are:

    printf("Hello, World!");
    int x = 6;
  • Semicolon (;)
    • In C we use a semi colon at the end of each command to indicate to the computer that its the end of the command
