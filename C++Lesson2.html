Lesson #2 :Anatomy of a Simple C++ Program:

In this lesson we are going to dive deep into the various components of a simple C++ program like that which displayed "Hello WOrld" in our previous lesson



A basic "Hello Wrold " program in C++ has a basic structure as follows

Hello World

1.#include <iostream>  
2.using namespace std;   
3.int main() {
4.  //Outputs Hello word
5.std::cout <<"Hello, World!" <<std::endl;
6.return 0;
7.}

Various Parts

Here's a breakdown of a basic "Hello, World!" program in C++ and its key components:

  1. Preprocessor Directive:

    2. #include <iostream> : This line instructs the compiler to include the iostream header file. The iostream header provides standard input/output objects like cin (for input) and cout (for output). Header files contain pre-written code that you can use in your program.

  2. Namespace Directive (Optional):

    3. using namespace std;: This line brings all the elements of the std (standard) namespace into the current scope.

    The std namespace contains many commonly used elements, including cout, cin , and other standard library components.

    Using this directive allows you to use these elements without the std:: prefix (e.g., cout instead of std::cout ).

  3. Main Function:

    4. int main(): This is the entry point of the program.

    int: Specifies that the main function returns an integer value.

    main(): The name of the function. The code within the curly braces {} is the body of the main function, where the program's logic is executed.

  4. Comment

    5. Comments are used to explain what the code does... single line comments begin with // and span a single line. Comments are ignored by the compiler

  5. Output Statement:

    6. cout << "Hello, World!" << endl;: This line prints the text "Hello, World!" to the console.

    cout: The standard output stream object.

    <<: The stream insertion operator, used to send data to the output stream.

    "Hello, World!": The string literal to be printed.

    endl: A manipulator that inserts a newline character, moving the cursor to the next line.

  6. Return Statement:

    7. return 0;: This statement indicates that the program executed successfully. The value 0 is conventionally used to signify successful execution.
