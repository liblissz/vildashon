Lesson #3 :Comments in C++ :

In this lesson, we are going to learn about comments and their uses in any C++ program

Overview

Comments are non-executable lines of text that are ignored by the compiler. They are used to provide explanations or notes within the source code to improve readability and understanding for human developers.



Types of Comments

  1. Single Line Comments

    • Begin with "//" and continue until the end of the line.
    • Example:

      • 1. //This is a single line comment

  2. Multi Line Comments

    • Begin with /* and end with */.
    • span multiple lines.
    • Example:

      •  1. /*This 
 2.    is  a Multi
 3.    line 
 4.    comment 
 5.*/


      Sample Code

      #include<iostream.h> 
using namespace std;
int main() { 
// declaration  and intialisation
int x = 3 ;
int y = 2 ;
/* 
  sum is gotten by adding  
  x with value of 3  
  and y with value of 2 
    */
int sum = x +y ;
std::cout <<"sum is" << sum << std::endl;
return 0;
}

      Output

      sum is 5


      Explanation

      In this example, the comments explain the purpose of the code and provide additional information for better understanding.



Purpose of Comments:

  • Explain the code:

    • Clarify complex logic, algorithms, or specific parts of the code.

  • Document the code:

    • Provide information about the author, creation date, modifications, and overall purpose of the code.

  • Temporarily disable code:

    • Comment out blocks of code for testing or debugging purposes.

Best Practices

  • Use comments sparingly and judiciously.

  • Write clear and concise comments.

  • Avoid redundant comments that merely restate the obvious.

  • Update comments when the code is modified.
