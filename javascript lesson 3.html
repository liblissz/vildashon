Lesson #3 :JavaScript BOM(Browser object Model):

  • Window object

    • The window object represents the brower window or tab.It provides access to various properties and methods for interacting with the browser window.

// Open a new browser window
let newWindow = window.open("https://example.com")
//Get the window's height and width
let windowWidth = window.innerWidth;
let widowHeight =
window.innerHeight;

  • History Object

    • The History object allows you to manipulate the browser's session history,enabling navigation through previosly visited pages.

// Go back in history
history.back();
// Go forward in history
history.forward();



  • Navigator Object

    • The Navigator object provides information about the browser,\ such as its name,version,and user agent.

// Get the browser name and version
let browserName= navigator.appName; let browserVersion =
navigator.appVersion;

  • Screen Object

    • The screen objects represents the user's screen,providing information about screen size and available screen properties.

// Get the screen's width and height
let screenWidth = screen.width; let screenHeigth = screen.height;
