Lesson #6 : Output C++:

In this lesson , we are going to learn how to print output to our console or computer screen

SubLesson #1 :Print Text

To print simple text in c, we will use the cout object together with the << operator , it is used to output values and print text.

When printing plain text , the intended text must be surounded with single codes (" "),followed by a semi colon ;

Sample Code

#include <iostream > 
int main(){
    cout << "Hello World" ;
    cout << "I am learning C++" ;
    return 0;
}

Output

Hello World!I am learning c++


Explaination

when ever the cout is written, it prints the content written in bracket, however the contents of the code will not be on seperate lines






Sub lesson #2 : Print Numbers

You can also use cout() to print numbers.

However unlike text, it should not be contained in double quotes:

Sample Code

#include <iostream > 
int main(){
    cout << 100 ;
    return 0;
}

Output

100

Explaination

when ever the cout is written, it prints the numbers here it can also be used to perform calculations however the contents of the code will not be on seperate lines




Sub lesson #3 : Print New Line

To insert new lines into your output, you can use the \n character

You can also use cout() to print numbers.

However unlike text, it should not be contained in double quotes:

Sample Code

#include <iostream > 
int main(){
    cout << "Hello world\n";
    cout << "I am learning c++";
    return 0;
}

Output

Hello world !
I am learning

Explaination

Due to the fact that we used \n, a new line was created after the first line which contained "hello world".
