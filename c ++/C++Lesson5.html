Lesson #5 : Seting up C++ Development Environment:

In this lesson, we are going to learn about how to set c++ development environment on our local computer

Algorithm


  1. Choose Your Operating System
    • Windows:

      • Widely used, offers a user-friendly environment.

    • macOS:

      • Known for its stability and developer-friendly tools.

    • Linux:

      • Highly customizable and popular among developers, often free and open-source.

  2. Install a C++ Compiler
    • Windows:

      • Visual Studio:

      A powerful IDE with a C++ compiler. Install the "Desktop development with C++" workload during installation.

      MinGW (Minimalist GNU for Windows):

      A free and open-source compiler suite.

    • macOS:

      • Xcode:

      Apple's integrated development environment, includes a C++ compiler.

      Homebrew:

      A package manager that can be used to install GCC (GNU Compiler Collection).

    • Linux:

      • GCC (GNU Compiler Collection):

      Usually pre-installed on most Linux distributions.

  3. Select an Editor or IDE (Integrated Development Environment)

    • Editors:
      • Visual Studio Code:

      A lightweight and versatile code editor with excellent C++ support through extensions.

      Sublime Text:

      A powerful and customizable text editor.

      Notepad++ (Windows):

      A simple and lightweight text editor.

    • IDEs:

      • Visual Studio (Windows):

      A full-featured IDE with debugging, code completion, and many other features.

      CLion (JetBrains):

      A powerful and intelligent C++ IDE.

      Code::Blocks:

      A free and open-source IDE specifically designed for C++ development.

  4. Install Necessary Tools (Optional)
    • Git:

      • A version control system for tracking changes to your code.

    • CMake:

      • A cross-platform build system that simplifies the process of building and testing your C++ projects.

    5. Verify Your Installation

    Create a simple "Hello, World!" program:

    C++ 
    #include < iostream >

int main() {
  std::cout << "Hello, World!"<< std::endl;
  return 0;
}
    Compile and run the program:
    • Using the command line:
      • GCC/G++:

      g++ hello.cpp -o hello ./hello

      Within your IDE:

      Use the build and run commands provided by your IDE.

  5. Explore and Customize
    • Customize your editor or IDE:

      • Configure settings for themes, fonts, code completion, and other preferences.

    • Install and explore useful extensions:

      • For example, in VS Code, install the "C/C++" extension for enhanced C++ support.

  6. Example:Setting up with Visual Studio Code on Windows

    1. Visual Studio Code: Download and install from https://code.visualstudio.com/

    2. the "C/C++" extension: Open VS Code, search for "C/C++" in the Extensions view, and install the extension by Microsoft.

    3. a C++ compiler: Download and install MinGW.

    4. Configure VS Code:

    5. In the "Command Palette" (Ctrl+Shift+P), type "C/C++: Edit Configurations (UI)".

    6. Select "Add Configuration" and choose "gcc.json".

    7. In the compilerPath setting, specify the path to your MinGW gcc executable (e.g., C:\MinGW\bin\gcc.exe).

    8. Save the configuration.
