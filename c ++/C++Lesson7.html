Lesson #7 Variables in C++:

In this lesson , we are going to learn about variables and how to declare and initialise them

OverView

A variable is a container in a program, designed to contain varaibles of specific data types some of this data types are :

Review

  • int:

    • used to store positve and negative whole numbers

  • float

    • used to store numbers with single precision such as 4.6600

  • double

    • Similar to float , but adds more precision for instance 4.60000000

  • char

    • Used to store single characters such as 'A'

  • String

    • Used to store text such as "hello"

  • bool

    • Used to store values with two states, true or false

SubLesson #1 : Declaring Variables

Declaration of variable is a common term, used to describe the creation of a variable

To create a variable , first we definr the type of data that the variable is goinging to contain , followed by the varaible name

  • Syntax
    •                             type variableName ;

    Sample Code

    #include <iostream > 
int main(){
    //declaration of an int with variable name age
    int age;
    cout << age ;
    return 0;
                           }

    Output

    0

    Explaination

    when we define the data type followed by variable naem, we have declared the variable.

  • Note

    • When avariable is defined without assignment , c++ gives it a default value which in the case of int is 0 .

    a single data type can be used to declare all values of that type, but must be seperated with a comma (,)

    int age, numberOfChildren ;
  • Use Case

    • We declare a variable without assignment when do not yet know the value which it is going to contain









    Sub lesson #2 :Initialisation

    Intitialisation is the process of giving a value to a declared variable.This is usually completed by proceeding the variable name with an " = "

    Lets give some value to the variable we declared erlier

    Sample Code

    #include <iostream > 
int main(){
  //declaration 
  int  age;
  //makes the age ready to contain a whole number 
  //initialisation 
   age  = 18;
    cout << age ;
    return 0;
}

    Output

    18

    Explaination

    By giving our variable age a value of 8, we have initialised it to 8,this means that if it is not changed alon the way it will remain 8.If we want our variable to never be changed, we are going to preceed it with the key word const.
